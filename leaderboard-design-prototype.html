Design Specifications

📋 Design Details

Technical specifications and design tokens

Color Palette

  • Primary Gradient: #667eea → #764ba2
  • Success (Profit): #10b981
  • Danger (Loss): #ef4444
  • Gold Medal: #ffd700 → #ffed4e
  • Silver Medal: #c0c0c0 → #e8e8e8
  • Bronze Medal: #cd7f32 → #e9a76c

Typography

  • Font Family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
  • Page Title: 48px / Bold / Gradient Text
  • Section Title: 24px / Bold
  • Body Text: 16px / Regular
  • Small Text: 14px / Medium

Spacing

  • Section Padding: 40px
  • Card Padding: 24-30px
  • Grid Gap: 24px
  • Button Padding: 12px 24px

Border Radius

  • Large Cards: 16px
  • Medium Components: 12px
  • Buttons: 8-10px
  • Avatars/Badges: 50% (Circle)

Animations & Transitions

  • Hover Effects: 0.3s ease
  • Transform Scale: translateY(-5px) on hover
  • Box Shadow: 0 10px 40px rgba(0,0,0,0.1)

Responsive Breakpoints

  • Mobile: < 768px (Card Layout)
  • Tablet: 768px - 1024px (Compact Table)
  • Desktop: > 1024px (Full Table Layout)

Key Features Demonstrated

  • ✅ Top 10/50 users leaderboard display
  • ✅ Time-based filters (Weekly, Monthly, All-Time)
  • ✅ Sort by Profit, Win Rate, and Predictions
  • ✅ Medal badges for top 3 positions
  • ✅ Clickable user rows linking to profile pages
  • ✅ User profile with comprehensive statistics
  • ✅ Responsive design (Desktop + Mobile views)
  • ✅ Dark mode support
  • ✅ Interactive hover states and animations
  • ✅ Statistics cards with key metrics

Accessibility Features

  • ✅ WCAG 2.1 AA compliant color contrast
  • ✅ Keyboard navigation support
  • ✅ Clear focus indicators
  • ✅ Semantic HTML structure
  • ✅ Readable font sizes (minimum 14px)
