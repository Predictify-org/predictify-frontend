Design Crit Frame Checklist — one-slide / frame to present

Use this checklist to build a single Figma frame or slide that captures the key items for the crit.

Top-left: Title + one-line goal
- "Streams list: Search + Filters v1 — goal: validate search scope, filters, and mobile sheet"

Center: Visuals
- Desktop primary frame (search bar, status chips, sort, list rows)
- Mobile frame or filter-sheet snapshot
- Make sure states shown: default, filtered, empty-results

Right: Actionable notes
- A11y calls: combobox ARIA, keyboard, focus state, live-region announcements
- Touch: confirm 44x44px min targets
- Copy: clarify search scope wording ("Search recipients")

Bottom: Ask & Decisions
- 1-line ask for product: Is search scope acceptable? (Yes/No)
- 1-line ask for eng: Any technical constraints for announcing counts or live regions?
- Decisions to capture: Soroban filtering deferred (v2), assets optional for v1

Footer: Next steps & owner
- Usability test (3 sessions) — owner: <name>
- WCAG self-check — owner: <name>
- Handoff exports — owner: <name>
